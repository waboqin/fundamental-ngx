Selector: fd-split-button-action-title

Directive used to identify the template which will populate the main action button. Used to achieve complex buttons that require more than a string.

   <fd-button-split>
       <ng-template fd-button-split-action-title>
           <p>Paragraph 1</p>
           <p>Paragraph 2</p>
       </ng-template>
       <div fd-button-split-menu>
           <fd-menu>
               <ul fd-menu-list>
                   <li fd-menu-item>
                      <a [routerLink]="'/'">option</a>
                   </li>
                   <li fd-menu-item>
                      <a [routerLink]="'/'">option2</a>
                   </li>
               </ul>
           </fd-menu>
       </div>
   </fd-button-split>
</fd-button-split>
