Selector: <fd-combobox>

Allows users to filter through results and select a value.

Supports Angular Forms.

<fd-combobox
     [(ngModel)]="searchTerm"
     [dropdownValues]="dropdownValues"
     [placeholder]="'Type some text...'">
</fd-combobox>
