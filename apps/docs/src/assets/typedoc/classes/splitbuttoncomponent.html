Selector: <fd-split-button>

Split Button component, used to enhance standard HTML button and add possibility to put some dropdown with additional options.

   <fd-split-button>
       Action Button
       <div fd-split-button-menu>
           <fd-menu>
               <ul fd-menu-list>
                   <li fd-menu-item>
                      <a [routerLink]="'/'">option</a>
                   </li>
                   <li fd-menu-item>
                      <a [routerLink]="'/'">option2</a>
                   </li>
               </ul>
           </fd-menu>
       </div>
   </fd-split-button>
